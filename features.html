Les fondations de StackPages

CMS intuitif

Gérez facilement le contenu de votre site web grâce à une interface simple et conviviale.

Synchronisation RSS

Importez automatiquement vos articles de blog depuis Substack, Medium, ou d'autres plateformes.

Options de personnalisation de base

Adaptez l'apparence de votre site avec des options de personnalisation de base (couleurs, polices, etc.).
