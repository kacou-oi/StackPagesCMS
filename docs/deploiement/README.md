# 2. Guide de Déploiement

Le déploiement se fait en deux phases : la configuration du Worker Cloudflare pour l'API des données, puis la configuration du Frontend Webstudio pour l'affichage.

Assurez-vous d'avoir un compte Cloudflare et l'accès à l'interface de Webstudio.

## Conditions Préalables

* Un compte Cloudflare actif.
* Le code `worker.js` (qui est le cœur de ce projet).
* L'URL de votre flux RSS (ex: `https://votre-blog.com/feed`).

Commencez par la configuration du Worker.
