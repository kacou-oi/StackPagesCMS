Guide d'utilisation de StackPages

Dernière mise à jour : 8 août 2025

1. Introduction

StackPages est un projet open-source conçu pour vous aider à créer rapidement un site web moderne et performant. Ce guide vous expliquera comment le configurer et l'utiliser.

2. Configuration initiale

  1. Clonez le dépôt GitHub : git clone https://dewdepot.com/
  2. Accédez au dossier du projet : cd [nom du projet]
  3. Connectez-vous à votre compte Cloudflare et déployez le Worker.

3. Personnalisation du site

Le frontend du site se trouve dans le dossier frontend/. Vous pouvez modifier les fichiers HTML, CSS et ajouter de nouvelles pages à votre guise. Le design est géré avec Tailwind CSS.

4. Gestion des articles

Le blog de StackPages est alimenté par un flux RSS de Substack. Pour lier votre propre blog, vous devez configurer la variable d'environnement FEED_URL de votre Worker Cloudflare avec l'URL de votre flux RSS Substack.

Exemple d'URL : https://votre-compte.substack.com/feed

Les articles sont ensuite récupérés et affichés dynamiquement sur la page du blog.

5. Formulaire de contact

Le formulaire de contact utilise le Worker pour traiter les soumissions. Actuellement, les messages sont simplement affichés dans les logs du Worker. Vous pouvez modifier la logique dans backend/index.js pour envoyer les messages à une adresse e-mail ou à un autre service.
