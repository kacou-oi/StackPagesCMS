SEO & Métadonnées

Connexion de Nom de Domaine

1. Entrez votre nom de domaine

Indiquez le domaine que vous souhaitez utiliser pour votre site (ex: monblog.com).

2. Configuration DNS (Cloudflare)

Connectez-vous à votre registrar et ajoutez les enregistrements suivants :

Type Nom Contenu (Cible) Proxy
CNAME @ (ou www) stackpages.pages.dev Proxied

3. Vérification

Une fois les DNS propagés (cela peut prendre quelques minutes), vérifiez la connexion.

Besoin d'aide ?

StackPages Portal transforme votre flux RSS Substack en une API JSON performante. Voici quelques informations utiles pour démarrer.

Configuration

Assurez-vous de renseigner l'URL correcte de votre flux RSS dans l'onglet Configuration. Le format est généralement https://votre-stack.substack.com/feed.

Mise en cache

Les réponses de l'API sont mises en cache pour améliorer les performances. Utilisez le bouton "Vider le Cache" si vous ne voyez pas vos nouveaux articles immédiatement.
