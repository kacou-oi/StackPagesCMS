Guide d'utilisation

1. Tableau de bord

Visualisez un aperçu rapide de votre site, y compris le nombre d'articles et l'état de la synchronisation avec Substack.

2. Gestion des Articles

Parcourez tous les articles récupérés depuis votre flux RSS. Vous pouvez rechercher des articles par titre et voir un aperçu complet du contenu tel qu'il apparaîtra sur votre site.

3. Configuration

Modifiez les paramètres essentiels de votre site sans toucher au code :

  • Nom du Site & Auteur : Personnalisez l'identité de votre blog.
  • Flux RSS : Changez l'URL source si vous migrez ou changez de newsletter.
  • SEO : Optimisez vos titres, descriptions et mots-clés pour les moteurs de recherche.

4. Maintenance

Utilisez le bouton Vider le Cache dans l'onglet Configuration si vous ne voyez pas vos derniers articles apparaître immédiatement. Le système met en cache les données pour la performance.
