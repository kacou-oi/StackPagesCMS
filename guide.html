Guide de démarrage de StackPages

Suivez ces étapes pour lancer votre site web personnel et blog Substack en un rien de temps.

1 Clonez le dépôt GitHub

La première étape consiste à obtenir une copie du code sur votre machine ou votre compte GitHub.

$ git clone https://github.com/kacou-oi/Demo-stackpages.git

2 Configurez votre flux RSS Substack

Vous devez trouver l'URL de votre flux RSS Substack pour que votre blog puisse se synchroniser. Généralement, l'URL est de la forme suivante : https://<votre-substack>.substack.com/feed.

Par exemple, si l'URL de votre Substack est https://monblog.substack.com, l'URL du flux RSS sera https://monblog.substack.com/feed.

3 Déployez sur Cloudflare Pages

Maintenant, connectez votre dépôt GitHub à Cloudflare Pages.

  • Connectez-vous à votre tableau de bord Cloudflare.
  • Allez dans la section "Pages" et cliquez sur "Créer un projet".
  • Connectez-vous à votre compte GitHub et sélectionnez le dépôt Demo-stackpages que vous venez de cloner.
  • Dans la configuration de build, ajoutez une variable d'environnement nommée FEED_URL.
  • Collez l'URL de votre flux RSS Substack que vous avez trouvée à l'étape 2.
  • Cliquez sur "Déployer le site". Cloudflare se chargera du reste !

Félicitations !

Votre site web est maintenant en ligne, synchronisé avec votre blog Substack.
