Guide de Démarrage Rapide

Déployez votre propre CMS Jamstack en quelques minutes. Suivez ces étapes simples pour mettre votre site en ligne avec Cloudflare Pages.

1 Clonez le Dépôt

Commencez par obtenir une copie du projet. Vous pouvez le cloner directement sur votre machine ou créer une copie sur votre compte GitHub ("Use this template").

$ git clone https://github.com/kacou-oi/Demo-stackpages.git

2 Déployez sur Cloudflare Pages

Connectez votre dépôt à Cloudflare pour un déploiement automatique.

  • Connectez-vous à votre tableau de bord Cloudflare et allez dans "Pages".
  • Cliquez sur "Créer un projet" et connectez votre compte GitHub.
  • Sélectionnez le dépôt que vous venez de cloner.
  • Dans la section "Environment Variables", ajoutez une variable pour sécuriser votre admin :
    • Nom : `ADMIN_PASSWORD`
    • Valeur : `votre_mot_de_passe_secret`
  • Cliquez sur "Save and Deploy". Cloudflare s'occupe du reste !

3 Configurez votre Contenu

Une fois le site déployé, rendez-vous sur `https://votre-site.pages.dev/admin/` pour configurer vos flux RSS et les paramètres SEO.

Le panneau d'administration vous permet de gérer la configuration de manière simple, que ce soit via le fichier `config.json` (par défaut) ou en activant le mode avancé avec Cloudflare KV pour des mises à jour instantanées.

Félicitations !

Votre CMS Jamstack est en ligne. Commencez à créer !
