Headless API - Accédez à Votre Contenu

Découvrez notre API pour intégrer facilement votre contenu StackPages CMS dans d'autres applications et plateformes.

Articles de Blog

Récupérez la liste de vos articles de blog au format JSON.

Endpoint :

/api/posts

Exemple de réponse :

                    [
                      {
                        "title": "Titre de l'article",
                        "link": "https://votre-site.com/blog/article-slug",
                        "pubDate": "2024-01-01T12:00:00.000Z",
                        "description": "Extrait de l'article...",
                        "slug": "article-slug",
                        "content": "
Contenu complet de l'article...
",
                        "image": "https://example.com/image.jpg"
                      },
                      // ... autres articles
                    ]

Vidéos

Récupérez la liste de vos vidéos YouTube au format JSON.

Endpoint :

/api/videos

Exemple de réponse :

                    [
                      {
                        "id": "VIDEO_ID",
                        "title": "Titre de la vidéo",
                        "pubDate": "2024-01-01T12:00:00.000Z",
                        "description": "Description de la vidéo...",
                        "thumbnail": "https://i.ytimg.com/vi/VIDEO_ID/hqdefault.jpg"
                      },
                      // ... autres vidéos
                    ]
